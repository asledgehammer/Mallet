Mallet BETA

                    
                

                

                    
Hello,

                    

                        Mallet is a program for documenting code in Project Zomboid. Mallet produces
                        JSON documents for the PZ-Rosetta framework,
                        deployed through the Umbrella project.

                        Mallet is currently a WIP tool and is being actively developed and polished.
                        Keep in mind that there may be jerky animations (or the lack thereof), when operating the tool.
                        Value checks and error-broadcasting are currently very limited. If you need to check and see if
                        the tool had an error occur, use the debug console in your web browser to check for errors.
                    

                    
Cheat Sheet:

                    
    
                        
  • Lua Class - A pseudo-class in Lua with properties. This is the majority of Lua files in
                            Project Zomboid's codebase.
    • 
                        
  • Lua Value - A static field in Lua. Can be assigned to classes, tables.
    • 
                        
  • Lua Field - A instance field in Lua. Can be assigned to classes. E.G: self.javaObject
    • 
                        
  • Lua Function - A static code-block in Lua that can execute. Can be assigned to Lua classes,
                            tables.
    • 
                        
  • Lua Method - A instance code-block in Lua that can executre. Can only be assigned to
                            classes. Runs with the class self
                            reference passed.
    •
